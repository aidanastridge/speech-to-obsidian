### Speech to Obsidian

#### Thesis

Records and transcribes daily-note into Obsidian vault.

#### Libraries
- [sounddevice]() 
- [wavio]()
- [DateTime]()
- [openai-whisper]()
- [json]()
- [markitdown]()

#### Order of Operations

1. Record
2. Export to WAV
3. Transcribe
4. Export to JSON
5. Convert to Markdown
6. Places it in Obsidian vault.
